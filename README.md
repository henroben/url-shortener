# React URL Shortener

Demo available at [https://henroben.github.io/url-shortener/](https://henroben.github.io/url-shortener/)

### Development Mode ###

Development server located on http://localhost:3000

```
	> npm install
	> npm start
```

### Testing ###

```
	> npm test
```

### Build for deployment ###

Deployment files located in /dist/

```
	> npm run build
```
